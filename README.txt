# Tutedude Task 2 - Basic Laundry Website

## Project Overview
This project is a Basic Laundry Service Website developed using HTML . The website provides information about laundry services, pricing and contact details in a clean and user-friendly interface.

## Features
- Responsive and clean design
- Home section with service introduction
- Services offered by the laundry business
- Pricing section
- Book Now section

## Technologies Used
- HTML5

## Project Structure
├── index.html
├── assets
└── README.txt


## Purpose
This project was created as part of the Tutedude Full Stack Development with MERN Task 2 assignment to demonstrate basic web development skills using HTML .
